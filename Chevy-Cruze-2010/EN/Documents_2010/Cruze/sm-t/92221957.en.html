DTC Descriptor

DTC P0630 00: VIN Not Programmed or Mismatched - Engine Control Module (ECM)

Circuit/System Description

This DTC sets when the engine control module (ECM) has determined that the vehicle identification number (VIN) is not stored in the ECM memory. When the control module is replaced, the VIN must be programmed into the control module memory.

Conditions for Running the DTC
    • The control module is ON or awake.
    • The DTC runs continuously when the above condition is met

Conditions for Setting the DTC

The ECM detects that no VIN is stored in memory.

Action Taken When the DTC Sets

DTCs P0630 00 is a Type C DTCs.

Conditions for Clearing the DTC

DTCs P0630 00 is a Type C DTCs.

Circuit/System Verification

Note: The ECM cannot determine if the VIN programmed into the control module is valid. Always verify that the VIN displayed on the scan tool is correct for that vehicle.

  1. Ignition ON, observe the Vehicle Information under ID System Information on the scan tool. The VIN information should match the vehicles VIN.
    2. ⇒If a VIN is not displayed, program the VIN with a scan tool.
    ⇒If a VIN is displayed, clear the DTCs with a scan tool. If DTC P0630 00 sets after clearing the DTCs, replace the K20 Engine Control Module.
  2. Operate the vehicle within the conditions for running the DTC. You may also operate the vehicle within the conditions that you observed from the freeze frame/failure records data.
