Engine Flywheel Replacement

Special Tools

    • EN-45000 Remover Oil Seal
    • EN-48251 Installer Oil Seal
    • EN-652 Locking Tool

For equivalent regional tools, refer to Special Tools .

Removal Procedure
  1. Remove the transmission:
    2. • For automatic transmission refer to Transmission Replacement .
    • Remove the torque converter. Refer to Torque Converter Removal .
    • For manual transmission refer to Transmission Replacement .
    • Remove the clutch refer to Clutch Pressure and Driven Plate Replacement .


  2. Install EN-652 locking tool  (1) to fix the engine flywheel.


  3. Remove the 6 engine flywheel bolts (2).
  4. Remove the engine flywheel (1).
  5. Check the crankshaft rear oil seal (1) replace if necessary.


    6. Note: Be careful do not damage the sealing surfaces.

  6. Use EN-45000 remover  (1) to remove the crankshaft rear oil seal (2).
  7. Remove the crankshaft rear oil seal (2).

Installation Procedure


  1. Use the EN-48251 installer  (2) to install a NEW crankshaft rear oil seal (1).
  2. Install the NEW crankshaft rear oil seal (1).


  3. Install the engine flywheel (1).

    4. Note: Clean thread before insert bolt with locking compound (red). The maximum installation time including torque check is 10 minutes.

  4. Install the 6 engine flywheel bolts (2).


  5. Install EN-652 locking tool  (1) to fix the engine flywheel.

    6. Caution: Refer to Fastener Caution in the Preface section.

  6. Tighten the 6 engine flywheel bolts to 105 N·m (78 lb ft).
  7. Install the torque converter. Refer to Torque Converter Installation .
  8. Install the transmission. Refer to Transmission Replacement .
  9. Install the transmission:
    10. • For automatic transmission refer to Transmission Replacement .
    • Install the torque converter. Refer to Torque Converter Removal .
    • For manual transmission refer to Transmission Replacement .
    • Install the clutch refer to Clutch Pressure and Driven Plate Replacement .
