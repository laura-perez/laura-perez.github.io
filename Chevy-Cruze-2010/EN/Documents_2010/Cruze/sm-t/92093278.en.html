Circuit/System Description

The BCM monitors, receives and transmits serial data to each of the doors switches and window motors. When a window switch has been activated on the driver window switch, a signal is sent to the BCM which sends a signal to the appropriate window motor and the window motor will move in the direction commanded.

Conditions for Running the DTC
    • Battery voltage must be between 9-16 volts.
    • The ignition switch is in the ON position

Conditions for Setting the DTC

B3205 00, B3210 00, B3215 00, B3220 00

The BCM detects a fault on a window motor control circuit.

B3205 39, B3210 39, B3215 39, B3220 39

An internal malfunction or a momentary voltage spike has been detected by the BCM in the power window motor while commanding the window UP or DOWN.

B3205 42, B3210 42, B3215 42, B3220 42

A not normalized/programmed door power window motor.

B3205 4B, B3210 4B, B3215 4B, B3220 4B

The window motor experiences a loss of voltage in the B+ circuit or an open/high resistance in the serial data circuit.

Action Taken When the DTC Sets

The affected power window will malfunction.

Conditions for Clearing the DTC

B3205 00, B3210 00, B3215 00, B3220 00

    • The DTC will be current for as long as the fault is present.
    • The DTC will change from current to history when the fault is no longer present.
    • A history DTC will clear after 100 consecutive ignition cycles if the condition for the malfunction is no longer present.

B3205 39, B3210 39, B3215 39, B3220 39

If the DTC is current, the appropriate window motor must be replaced before the DTC can be cleared.

B3205 42, B3210 42, B3215 42, B3220 42

The affected power window motor must be programmed before the DTC can be cleared.

B3205 4B, B3210 4B, B3215 4B, B3220 4B

The affected power window motor must be calibrated before the DTC can be cleared.

Circuit/System Verification

Ignition ON, observe the operation of each power window while commanding each window UP and DOWN with a scan tool. Each power window should go UP and DOWN command.

Circuit/System Testing

B3205 00, B3210 00, B3215 00, B3220 00

  1. Ignition OFF, disconnect the harness connector at the appropriate M74 window motor.
  2. Test for less than 5 ohms between the ground circuit terminal 1 and ground.
    3. ⇒If greater than the specified range, test the ground circuit for an open/high resistance.
  3. Ignition ON, verify that a test lamp illuminates between the B+ circuit terminal 2 and ground.
    4. ⇒If the test lamp does not illuminate, test the B+ circuit for a short to ground or an open/high resistance.
  4. Connect a test lamp between the control circuit terminal 3 and ground.
  5. Command the power window UP by using the appropriate S79 window switch. The test lamp should illuminate during the UP command.
    6. ⇒If the test lamp is always OFF, test the control circuit for a short to ground or an open/high resistance. If the circuit tests normal, replace the S79 window switch.
    ⇒If the test lamp is always ON, test the control circuit for a short to voltage. If the circuit tests normal, replace the S79 window switch.
  6. Connect a test lamp between the control circuit terminal 7 and ground.
  7. Command the power window DOWN by using the S79 window switch. The test lamp should illuminate during the DOWN command.
    8. ⇒If the test lamp is always OFF, test the control circuit for a short to ground or an open/high resistance. If the circuit tests normal, replace the S79 window switch.
    ⇒If the test lamp is always ON, test the control circuit for a short to voltage. If the circuit tests normal, replace the S79 window switch.
  8. Connect a test lamp between the control circuit terminal 5 and ground.
  9. Command the power window to EXPRESS DOWN by using the S79 window switch. The test lamp should illuminate during the EXPRESS DOWN command.
    10. ⇒If the test lamp is always OFF, test the control circuit for a short to ground or an open/high resistance. If the circuit tests normal, replace the S79 window switch.
    ⇒If the test lamp is always ON, test the control circuit for a short to voltage. If the circuit tests normal, replace the S79 window switch.
  10. Ignition OFF, connect the harness connector at the M74 window motor.
  11. Ignition ON, command the power window UP and DOWN with a scan tool. The power window should perform the UP and DOWN function.
    12. ⇒If the window does not perform as specified, replace the M74 window motor.
  12. If all circuits test normal, replace the M74 window motor

B3205 39, B3210 39, B3215 39, B3220 39

Ignition ON, command the appropriate power window DOWN and UP with a scan tool. The power window should go DOWN and UP as commanded.

⇒If the DTC is current and the power window is inoperative, replace the M74 window motor.
⇒If the DTC is history and the power window performs the UP and DOWN functions, clear the DTC.

B3205 42, B3210 42, B3215 42, B3220 42

Reprogram the appropriate M74 window motor with a scan tool and operate the power window within the conditions for running the DTC, verify the DTC does not reset.

⇒If the DTC resets, replace the M74 window motor.

B3205 4B, B3210 4B, B3215 4B, B3220 4B

  1. Ignition ON, command the appropriate power window fully UP by lifting and holding the appropriate S79 window switch for 5 seconds
  2. Command the power window fully DOWN by pressing and holding the S79 window switch for 5 seconds.
  3. Command the express up and express down functions of the window by pressing the S79 window switch past the first detent and releasing and pulling the S79 window switch past the first detent and releasing. The power window should perform the express up and express down function.
    4. ⇒If the DTC resets, replace the M74 window motor.
