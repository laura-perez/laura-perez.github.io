Mode Valve Lever Replacement (LHD)

Removal Procedure
  1. Remove instrument panel compartment assembly. Refer to Instrument Panel Compartment Replacement .


  2. Remove instrument panel wiring harness (2) from instrument panel tie bar (1).

    3. Disconnect electrical connectors.



  3. Remove instrument panel tie bar (2).
    4. • Remove 4 instrument panel tie bar bolts (3, 4).
    • Remove 2 instrument panel tie bar nuts (1).


  4. Remove mode control cam (1).


  5. Remove 2 mode valve lever (1).

Installation Procedure


    Note: The mode valve levers have to be fitted to the according mode valve gears in figured position. The ventilation dampers have to be in middle position.

  1. Install 2 mode valve lever (1).


    2. Note: The mode control cam can only be fitted in one position.

  2. Install mode control cam (1).

    3. Caution: Refer to Fastener Caution in the Preface section.



  3. Install instrument panel tie bar (2).
    4. • Install 4 instrument panel tie bar bolts (3 4) and tighten to  XXX N·m (XXX lb in).
    • Install 2 instrument panel tie bar nuts (1) and tighten to  XXX N·m (XXX lb in).


  4. Install instrument panel wiring harness (2) from instrument panel tie bar (1).

    5. Connect electrical connectors.

  5. Install instrument panel compartment assembly. Refer to Instrument Panel Compartment Replacement .
