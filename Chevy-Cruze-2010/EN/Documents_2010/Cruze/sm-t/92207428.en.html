1. Power Door Latch Malfunction

Circuit/System Description

The passenger door lock switch signal circuit provides input to the remote control door lock receiver when the right front exterior door handle switch is activated. This input allows the remote control door lock receiver to detect a door lock request upon exit of the vehicle. The remote control door lock receiver provides voltage to the passenger door lock switch via the passenger door lock switch signal circuit. When the passenger door lock is activated, the switch closes and the voltage signals from the remote control door lock receiver are pulled to ground.

Conditions for Running the DTC

Battery voltage is between 9-16 V.

Conditions for Setting the DTC

The remote control door lock receiver detects the passenger door lock switch signal circuit is shorted to ground or closed for more than 45 s.

Action Taken When the DTC Sets

The remote control door lock receiver ignores inputs from the right front exterior door handle switch.

Conditions for Clearing the DTC
    • The DTC will be current for as long as the fault is present.
    • When the fault is no longer present, the DTC will be a history DTC.
    • A history DTC will clear after 50 ignition cycles.

Circuit/System Verification

Ignition ON, observe the scan tool Exterior Passenger Door Handle Proximity Sensor parameter while using the passenger door handle. The parameter should change between Active and Inactive.

Circuit/System Testing
  1. Ignition OFF, disconnect the harness connector at the T10N Keyless Entry Antenna - Passenger Door Handle.
  2. Ignition ON, verify the scan tool Exterior Passenger Door Handle Proximity Sensor parameter is Inactive.
    3. ⇒If not the specified value, test the signal circuit terminal 3 for a short to ground. If the circuit tests normal, replace the K77 Remote Control Door Lock Receiver.
  3. Connect the a 3 A jumper harness between the signal circuit terminal 3 and ground. Verify the scan tool Exterior Passenger Door Handle Proximity Sensor parameter is Active.
    4. ⇒If not the specified value, test the signal circuit terminal 3 for a open/high resistance. If the circuit tests normal, replace the K77 Remote Control Door Lock Receiver.
  4. If all circuits test normal, replace the T10N Keyless Entry Antenna - Passenger Door Handle.
