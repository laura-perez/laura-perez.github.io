Front Compartment Fuse Block Housing Replacement

Removal Procedure
  1. Disconnect battery negative cable. Refer to Battery Negative Cable Disconnection and Connection .
  2. Open front compartment fuse block.


  3. Remove 3 front compartment fuse block screw (1).
  4. Remove fuse plate from front compartment fuse block.
  5. Remove windshield washer solvent container filler tube.


  6. Remove plate (1) from front compartment fuse housing block.


  7. Remove nut (3) of wiring harness (2) and remove wiring harness.
  8. Remove front compartment fuse block housing.
    9. • Remove 3 connector from front compartment fuse block housing
    • Remove attaching parts

Installation Procedure
  1. Install front compartment fuse block housing.
    2. • Install attaching parts
    • Install 3 connector

    Caution: Refer to Fastener Caution in the Preface section.

  2. Install wiring harness and install nut, tighten to 22 N·m (17 lb ft).
  3. Install plate.
  4. Install windshield washer solvent container filler tube.
  5. Install fuse plate.
  6. Install 3 front compartment fuse block screw, tighten to 7 N·m (62 lb in).
  7. Close front compartment fuse block.
  8. Connect battery negative cable. Refer to Battery Negative Cable Disconnection and Connection .
