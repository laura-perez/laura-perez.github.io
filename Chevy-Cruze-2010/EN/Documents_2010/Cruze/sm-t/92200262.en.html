Circuit/System Description

When the automatic transmission is placed in the reverse position, the transmission control module (TCM) sends a serial data message to the body control module (BCM). The message indicates that the gear selector is in the reverse position. The BCM applies battery voltage to the backup lamps. The backup lamps are permanently grounded. Once the driver moves the gear selector out of the reverse position, a message is sent by the TCM via serial data requesting the BCM to remove battery voltage from the backup lamp control circuit.

Circuit/System Verification

Ignition ON, perform the Backup Lamps Relay test. The backup lamp should be ON/OFF.

Circuit/System Testing
  1. Observe the parameter Tap Up/Down Switch when putting the gear selector is in the reverse position. The parameter Tap Up/Down Switch should display that reverse gear is engaged.
    2. ⇒If the Tap Up/Down Switch parameter displays the corresponding gear, replace the K9 Body Control Module.
    ⇒If the Tap Up/Down Switch parameter does not displays the corresponding gear, refer to Transmission Control Lever Malfunction .
