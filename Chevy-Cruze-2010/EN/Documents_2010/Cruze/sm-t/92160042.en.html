DTC Descriptors

DTC B101E 43: ECU EEPROM Program Error

DTC B101E 4B: Calibration Not Learned

Circuit/System Description

When the ignition is turned ON, the passenger presence system performs a self test to verify if any critical malfunction exists within the system. During the power-up mode the passenger presence system checks for a valid calibration data set. If no valid calibration set is found the passenger presence system disables the system.

Conditions for Running the DTC

System voltage is between 9-16 V.

Conditions for Setting the DTC

The calibration data set in not programmed in the passenger presence system.

Action Taken When the DTC Sets

Passenger presence system functions are disabled.

Conditions for Clearing the DTC
    • The condition for setting the DTC is no longer present.
    • The history DTC clears after 100 fault-free ignition cycles.
