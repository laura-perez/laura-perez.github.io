Transmission Control Replacement

Removal Procedure
  1. Set the park brake and chock the wheels.
  2. Remove the console. Refer to Front Floor Console Replacement .


  3. Disconnect the transmission range selector lever cable terminal (2) from the shift lever pin (3).
  4. Release the transmission range selector lever cable retainer (1) and remove the transmission range selector cable from the transmission control lever base (4).


  5. Remove the transmission shift control bolts (1).
  6. Remove the transmission shift control (2).

Installation Procedure


  1. Install the transmission shift control (2).

    2. Caution: Refer to Fastener Caution in the Preface section.

  2. Install the transmission shift control bolts (1) and tighten to  8 N·m(71 lb in).


  3. Install the transmission range selector lever cable retainer (1) to the shift lever base (4).
  4. Install the transmission range selector lever cable terminal (2) to the shift lever pin (3).
  5. Adjust the automatic transmission range selector lever cable. Refer to Range Selector Lever Cable Adjustment .
  6. Install the console. Refer to Front Floor Console Replacement .
