Glass Breakage Sensor Malfunction

    • Perform the Diagnostic System Check - Vehicle prior to using this diagnostic procedure.

Circuit/System Description

The glass breakage sensors are located on the rear side windows and is supplied B+ at all times. The body control module grounds the glass breakage sensor circuit. In the event of the interruption of voltage to the glass breakage sensors, the alarm will sound.

Circuit/System Verification

Ignition ON, observe the scan tool Current Content Theft Deterrent Trigger parameter. The reading should be 0 counts.

Circuit/System Testing
  1. Ignition OFF, disconnect the harness connector at the B159L left glass breakage sensor.
  2. Ignition OFF, test for less than 10 Ω between the ground circuit terminal 1 X1 and ground.
    3. ⇒If greater than the specified range, test the ground circuit for an open/high resistance. If the circuit tests normal, replace the K9 BCM.
  3. Install a test lamp between the ground circuit terminal 1 X1 and the control circuit terminal 1 X2.
    4. ⇒If the test lamp is always OFF, test the control circuit for short to ground or an open/high resistance. If the circuit tests normal, replace the B159R right glass breakage sensor.
  4. If all circuits tests normal, test or replace the B159L left glass breakage sensor.
