Fuel Tank Fuel Pump Module Replacement

Special Tools

    • EN 6015 Plugs
    • EN-48279 Main Fuel Pump Locking Ring Remover/Installer

For equivalent regional tools, refer to Special Tools .

Removal Procedure
  1. Remove the fuel tank. Refer to Fuel Tank Replacement .


  2. Lower the fuel tank slowly down to a high until the fuel pump wiring harness plug (1) is accessible.
  3. Disconnect the fuel tank vent pipe (2) and fuel feed pipe (2) from the fuel tank fuel pump module (4).
  4. Disconnect fuel tank fuel pump module wiring harness (1) from the fuel tank fuel pump module (4).


  5. Install the EN-48253 remover/installer to the fuel pump module lock ring.

    6. Note: Do NOT use impact tools. Significant force will be required to release the lock ring. The use of a hammer and screwdriver is not recommended. Secure the fuel tank in order to prevent fuel tank rotation.

  6. Using the EN-48253 remover/installer and a long breaker-bar, rotate the lock ring in a counterclockwise direction in order to unlock the lock ring.


  7. Remove fuel tank fuel pump module lock ring (3).
  8. Remove the fuel tank fuel pump module (4)

    9. Note: Lift the fuel tank fuel pump module up slightly.

  9. Remove and discard the fuel pump module seal (5).

Installation Procedure


  1. Raise the fuel tank with a suitable adjustable jack to a high until the fuel pump wiring harness plug (1) is accessible.
  2. Install a NEW fuel pump module seal (5).
  3. Install the fuel tank fuel pump module (4)
  4. Install fuel tank fuel pump module lock ring (3).


  5. Install the EN-48253 remover/installer to the fuel pump module lock ring.

    6. Note: Do NOT use impact tools. Significant force will be required to release the lock ring. The use of a hammer and screwdriver is not recommended. Secure the fuel tank in order to prevent fuel tank rotation.

  6. Using the EN-48253 remover/installer and a long breaker-bar, rotate the lock ring in a clockwise direction in order to lock the lock ring.


  7. Connect fuel tank fuel pump module wiring harness (1) to the fuel tank fuel pump module (4).
  8. Connect the fuel tank vent pipe (2) and fuel feed pipe (2) to the fuel tank fuel pump module (4).
  9. Install the fuel tank. Refer to Fuel Tank Replacement .
