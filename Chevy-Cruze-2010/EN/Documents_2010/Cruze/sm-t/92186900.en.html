Heated Seat Module Scan Tool Data Parameters

Parameter

System State

Expected Value

Description

Operating Conditions: Ignition ON

Left Front Seat Cushion Heating Command

--

Inactive

The scan tool displays Inactive or Active. This parameter displays the commanded output from the heated seat module to the seat heater elements.

Left Front Seat Cushion Temperature Sensor

--

Varies

The scan tool displays 0 - 5 V. The voltage displayed is an input to the heated seat module from the seat cushion temperature sensor. Higher voltage readings indicate cooler seat temperatures while lower voltage readings indicate warmer seat temperatures.

Left Front Seat Heating/Venting/Cooling Mode

--

Off

The scan tool displays Off or Back & Cushion Heat. This parameter indicates the heated seat mode of operation.

Left Front Seat Heating/Venting/Cooling Level

--

Off

The scan tool displays Off, High Medium, or Low. When the heated seat is active, this parameter displays the selected temperature setting.

Right Front Seat Cushion Heating Command

--

The scan tool displays Inactive or Active. This parameter displays the commanded output from the heated seat module to the seat heater elements.

The scan tool displays 0 - 5 V. The voltage displayed is an input to the heated seat module from the seat cushion temperature sensor. Higher voltage readings indicate cooler seat temperatures while lower voltage readings indicate warmer seat temperatures.

--

Off

The scan tool displays Off or Back & Cushion Heat. This parameter indicates the heated seat mode of operation.

Right Front Seat Heating/Venting/Cooling Level

--

Off

The scan tool displays Off, High Medium, or Low. When the heated seat is active, this parameter displays the selected temperature setting.

Heated Seat Module Scan Tool Output Controls

Output Control

Description

Left Front Seat Cushion Heating

This output control is used to command the left front seat heater elements On.

Left Front Seat Blower

This output control is used to command the left front seat blower motor On.

Right Front Seat Cushion Heating

This output control is used to command the right front seat heater elements On.

Right Front Seat Blower

This output control is used to command the right front seat blower motor On.
