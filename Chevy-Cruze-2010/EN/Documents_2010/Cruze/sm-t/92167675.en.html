Circuit/System Description

When the transmission is placed in the reverse position, the engine control module (ECM) sends a serial data message to the body control module (BCM). The message indicates that the gear selector is in the reverse position. The BCM applies battery voltage to the reverse lamps. The reverse lamps are permanently grounded. Once the driver moves the gear selector out of the reverse position, a message is sent by the ECM via serial data requesting the BCM to remove battery voltage from the reverse lamp control circuit.

Circuit/System Verification

Ignition ON, perform the Reverse Lamps Relay Command test. The reverse lamp should be ON/OFF.

Circuit/System Testing
  1. Ignition OFF, disconnect the harness connector at the B16 Reverse Lamp Switch.
  2. Test for less than 5 Ω between the ground circuit harness connector terminal B and ground.
    3. ⇒If greater than the specified value, test the ground circuit for an open/high resistance.
  3. Disconnect the X2 harness connector at the K20 Engine Control Module (ECM).
  4. ignition On, test for less than 0.3 V at the B16 Reverse Lamp Switch signal circuit harness connector terminal A and ground.
    5. ⇒If greater then the specified value, test the signal circuit for a short to voltage.
  5. Ignition OFF, test for infinite resistance between the B16 Reverse Lamp Switch signal circuit harness connector A and ground.
    6. ⇒If lower then the specified value, test the signal circuit for a short to ground.
  6. Connect the harness connector at the B16 Reverse Lamp Switch.
  7. Reversing gear engaged, test for less than 85 kΩ between the K20 Engine Control Module (ECM) X2 harness connector terminal listed below and ground:
    8. • Terminal 87 with LXV
    • Terminal 40 with LLW
    ⇒If greater then the specified value, test the signal circuit for an open/high resistance. If the circuit test normal, test or replace the B16 Reverse Lamp Switch.
  8. Reversing gear disengaged, test for infinite resistance between the K20 Engine Control Module (ECM) X2 harness connector terminal listed below and ground:
    9. • Terminal 87 with LXV
    • Terminal 40 with LLW
    ⇒If lower then the specified value, test or replace the B16 Reverse Lamp Switch.
  9. If all the circuits test normal, replace the K20 Engine Control Module (ECM).

Component Testing

Static Test

  1. Ignition OFF, disconnect the harness connector at the B16 Reverse Lamp Switch. Reversing gear engaged, test for less than 80 kΩ at the B16 Reverse Lamp Switch terminal A and terminal B.
    2. ⇒If greater than the specified value replace the B16 Reverse Lamp Switch.
  2. Reversing gear disengaged, test for infinite resistance at the B16 Reverse Lamp Switch terminal A and terminal B.
    3. ⇒If less than the specified value replace the B16 Reverse Lamp Switch.
