Fuel Rail Installation



    Note: Coat the seal rings with lubricant.

  1. Install NEW injector seal rings.

    2. Note: Inspect the installation position and ensure correct fitting.

  2. Install the 4 injectors (2) to the fuel injection fuel rail (1).
  3. Install 4 retaining clamps (3).


    4. Note: Be careful while Installing. Do not damage the injectors and injector seal rings.

  4. Install the fuel injection fuel rail (2).

    5. Caution: Refer to Fastener Caution in the Preface section.

  5. Install the 2 fuel injection fuel rail bolts (1) and tighten to  25 N·m (18 lb ft).


  6. Install the intake manifold bracket (3).
  7. Install the 2 intake manifold bracket nuts (2) and tighten to  25 N·m (18 lb ft).
  8. Install the 2 intake manifold bracket bolts (1) and tighten to  25 N·m (18 lb ft).
