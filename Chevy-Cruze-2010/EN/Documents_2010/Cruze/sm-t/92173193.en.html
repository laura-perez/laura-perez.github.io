Circuit/System Description

When any exterior door handle is lifted, switched ground signals are sent to the remote control door lock receiver through the door handle switch signal circuits. Upon receipt of these inputs, the remote control door lock receiver will check the door lock status of the following inputs:

    • Door unlock switch activation
    • Keyless entry unlock command
    • Passive keyless entry unlock command message is received by the remote control door lock receiver

Once the enable criteria has been met, the remote control door lock receiver will supply power and ground to door catch control circuits and the door will release.

Conditions for Running the DTC

Battery voltage is between 9-16 V.

Conditions for Setting the DTC

B250B 01

The remote control door lock receiver detects a short to battery in the driver door uncatch motor uncatch control low circuit.

B250B 02

The remote control door lock receiver detects a short to ground in the driver door uncatch motor uncatch control low circuit.

Action Taken When the DTC Sets

The remote control door lock receiver ignores inputs from the driver exterior door handle switch.

Conditions for Clearing the DTC
    • The DTC will be current for as long as the fault is present.
    • When the fault is no longer present, the DTC will be a history DTC.
    • A history DTC will clear after 50 ignition cycles.

Circuit/System Verification

Ignition ON, command the Driver Door PEPS Release to Lock and Unlock. The driver door catch should Catch and Uncatch when changing between commanded states.

Circuit/System Testing
  1. Ignition OFF, disconnect the harness connector at the A23D Door Latch Assembly - Driver.
  2. Test for less than 10 Ω between the control circuit terminal 10 and ground.
    3. ⇒If greater than the specified range, test the control circuit for a short to voltage or an open/high resistance. If the circuit tests normal, replace the K77 Remote Control Door Lock Receiver.
  3. Test for less than 10 Ω between the control circuit terminal 5 and ground.
    4. ⇒If greater than the specified range, test the control circuit for a short to voltage or an open/high resistance. If the circuit tests normal, replace the K77 Remote Control Door Lock Receiver.
  4. Connect a test lamp between the control circuit terminals 5 and 10.
  5. Command the Driver Door PEPS Release to Lock and Unlock with a scan tool. The test lamp should turn ON and OFF when changing between the commanded states.
    6. ⇒If the test lamp is always OFF during either of the commands, test either control circuit for a short to ground. If the circuits test normal, replace K77 Remote Control Door Lock Receiver.
  6. If all circuits test normal, test or replace the A23D Door Latch Assembly - Driver.

Component Testing

Driver Door Catch Test

  1. Ignition OFF, disconnect the harness connector at the A23D Door Latch Assembly - Driver.
  2. Install a 15 A fused jumper wire between one of the control terminals and 12 V. Momentarily install a jumper wire between the other control terminal and ground. Reverse the jumper wires at least two times, the door catch should perform the Lock/Unlock functions.
    3. ⇒If it does not perform as specified, replace the A23D Door Catch Assembly - Driver.
