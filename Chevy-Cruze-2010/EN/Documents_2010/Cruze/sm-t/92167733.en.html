Circuit/System Description

The body control module (BCM) controls the all doors lock, passenger doors unlock and driver door unlock circuits. Voltage is applied to the appropriate circuit when the BCM senses a door lock/unlock switch activation or a keyless entry lock or unlock command.

Circuit/System Verification
  1. Command the All Doors Lock, Driver Door Unlock and Passenger Door(s) Unlock with the scan tool, the doors should lock and unlock.
    2. ⇒If the driver door does not unlock, refer to Driver Door Unlock Circuit Malfunction.
    ⇒If one or more passenger doors do not unlock, refer to Door Unlock Circuit Malfunction.
    ⇒If one or more vehicle doors do not lock, refer to Door Lock Circuit Malfunction.

Circuit/System Testing

Driver Door Unlock Circuit Malfunction

  1. Ignition OFF, disconnect the harness connector at the A23D Door Latch Assembly - Driver.
  2. Verify that a test lamp does not illuminate between the driver door unlock control circuit terminal 3 and ground.
    3. ⇒If the test lamp illuminates, test the control circuit for a short to voltage. If the circuit tests normal, replace the K9 Body Control Module (BCM).
  3. Test for less than 30 Ω between the driver door unlock and lock control circuit terminal 3 and 2.
    4. ⇒If greater than the specified range, test the driver door unlock and lock circuits for an open/high resistance. If both circuits test normal, replace the K9 Body Control Module (BCM).
  4. If all circuits test normal, test or replace the A23D Door Latch Assembly - Driver.

Door Unlock Circuit Malfunction

  1. Ignition OFF, disconnect the harness connector at the inoperative door catch.
  2. Verify that a test lamp does not illuminate between the door unlock control circuit listed and ground.
    3. • A23P Door Catch Assembly - Passenger terminal 8
    • A23LR Door Latch Assembly - Left Rear terminal 3
    • A23RR Door Latch Assembly - Right Rear terminal 8
    ⇒If the test lamp illuminates, test the control circuit for a short to voltage. If the circuit tests normal, replace the K9 Body Control Module (BCM).
  3. Test for less than 30 Ω between the door unlock and lock control circuit terminals listed.
    4. • A23P Door Catch Assembly - Passenger terminal 7 and 8
    • A23LR Door Latch Assembly - Left Rear terminal 2 and 3
    • A23RR Door Latch Assembly - Right Rear terminal 7 and 8
    ⇒If greater than the specified range, test the passenger door unlock and lock circuits for an open/high resistance. If both circuits test normal, replace the K9 Body Control Module (BCM).
  4. If all circuits test normal, test or replace the inoperative door latch.

Door Lock Circuit Malfunction

  1. Ignition OFF, disconnect the harness connector at the inoperative door catch.
  2. Ignition ON, test for 7.5-8.5 V between the door lock control circuit listed and ground.
    3. • A23D Door Catch Assembly - Driver terminal 2
    • A23D Door Catch Assembly - Driver terminal 3
    • A23P Door Catch Assembly - Passenger terminal 7
    • A23P Door Catch Assembly - Passenger terminal 8
    • A23LR Door Latch Assembly - Left Rear terminal 2
    • A23LR Door Latch Assembly - Left Rear terminal 3
    • A23RR Door Latch Assembly - Right Rear terminal 7
    • A23RR Door Latch Assembly - Right Rear terminal 8
    ⇒If greater than the specified range, test the control circuit for a short to voltage. If the circuit tests normal, replace the K9 Body Control Module (BCM).
    ⇒If less than the specified range, test the control circuit for an open/high resistance. If the circuit tests normal, replace the K9 Body Control Module (BCM).
  3. Connect a test lamp between the door lock control circuit listed.
    4. • A23D Door Catch Assembly - Driver terminal 2 and 3
    • A23P Door Catch Assembly - Passenger terminal 7 and 8
    • A23LR Door Latch Assembly - Left Rear terminal 2 and 3
    • A23RR Door Latch Assembly - Right Rear terminal 7 and 8
  4. Command the All Doors Lock, Driver Door Unlock and Passenger Door(s) Unlock with a scan tool. The test lamp should briefly illuminate while commanding the lock and unlock states
    5. ⇒If the test lamp remains OFF during either of the commands, replace the K9 Body Control Module (BCM).
  5. If all circuits test normal, test or replace the inoperative door latch.

Component Testing

Dynamic Test

  1. Ignition OFF, disconnect the harness connector at the appropriate door lock assembly.
  2. Install a 25 A fused jumper wire between the lock control terminal and B+. Momentarily install a jumper wire between the unlock control terminal and ground. The door lock should perform the lock/unlock function.
    3. ⇒If the function does not perform as specified, replace the door lock.
  3. Reverse the jumper wires; the appropriate door lock should perform the unlock/lock function.
    4. ⇒If the function does not perform as specified, replace the door lock.
