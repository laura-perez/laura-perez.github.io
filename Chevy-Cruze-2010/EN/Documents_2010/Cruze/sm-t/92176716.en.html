Timing Belt Rear Cover Installation

Special Tools

EN-45059 Torque Angle Sensor Kit

For equivalent regional tools, refer to Special Tools .

  1. Clean the 4 rear timing belt cover threads.
  2. Apply threadlock to the 4 NEW rear timing belt cover bolts. Refer to Adhesives, Fluids, Lubricants, and Sealers for the recommended thread locking material.


  3. Install the timing belt rear cover (1).

    4. Caution: Refer to Fastener Caution in the Preface section.

  4. Install the 4  timing belt rear cover bolts (2) and tighten to  6 N·m (53 lb in)


    5. Note: Two mechanics are required.

  5. Install the camshaft adjuster (3) and the camshaft adjuster bolts (2) and tighten the bolts in three passes using the EN-45059 sensor kit :

    6. Hold at the appropriate camshaft hexagon.

    5.1. First pass tighten to  65 N·m (48 lb ft)
    5.2. Second pass to  120 degrees
    5.3. Third pass to  15 degrees
  6. Install the 2 camshaft adjuster closure plugs (1) and tighten to  30 N·m (22 lb ft).
