Circuit/System Description

The instrument cluster illuminates the traction control/stability control-active indicator, winter indicator and stability control off indicator for 5 s after ignition ON. The electronic brake control module (EBCM) will request the instrument cluster via serial data to turn the traction control/stability control-active indicator ON (flashing) when the system is in traction control or stability control mode. When the EBCM detects a fault it sends a serial data message to the instrument cluster commanding the traction control/stability control-active indicator ON.

The body control module (BCM) monitors the stability control switch. When the stability control switch is pressed once, the BCM will request the EBCM via serial data to disable the traction control. When the stability control switch is pressed for 5 s, the BCM will request the EBCM to disable the stability control. The EBCM will request the instrument cluster via serial data to turn the winter indicator and stability control off indicator ON to notify the driver of the deactivation.

Circuit/System Verification
  1. Ignition ON, with a scan tool command the Instrument Cluster All Indicators test ON and OFF. Verify the traction control/stability control-active indicator, traction control off indicator and stability control off indicator turn ON and OFF.
    2. ⇒If either indicator does not turn ON and OFF, replace the P16 Instrument Cluster.
  2. Observe the scan tool BCM Traction Control Switch parameter while pressing and releasing the stability control switch. Verify the reading changes between Active and Inactive.
    3. ⇒If the parameter does not change between the specified values, refer to Circuit/System Testing.
  3. Observe the scan tool EBCM Traction Control Switch parameter while pressing and releasing the stability control switch. Verify the reading changes between Active and Inactive.
    4. ⇒If the parameter does not change between the specified values, replace the K9 Body Control Module.
  4. Observe the traction control off indicator while pressing and releasing the stability control switch. Verify the traction control off indicator turns ON and OFF.
    5. ⇒If the traction control off indicator does not turn ON and OFF, replace the K17 Electronic Brake Control Module (EBCM)
  5. Observe the stability control off indicator while pressing the stability control switch for 5 s and releasing. Verify the stability control off indicator turns ON and OFF.
    6. ⇒If the stability control off indicator does not turn ON and OFF, replace the K17 Electronic Brake Control Module (EBCM)

Circuit/System Testing

Note: Circuit/System Verification must be performed before Circuit/System Testing.

  1. Ignition OFF, disconnect the harness connector at the S48C Multi-Function Switch - Instrument Panel 1.
  2. Test for less than 5 Ω between the ground circuit terminal 4 and ground.
    3. ⇒If greater than the specified value, test the ground circuit for an open/high resistance.
  3. Ignition ON, verify the scan tool BCM Traction Control Switch parameter is Inactive.
    4. ⇒If not the specified value, test the signal circuit for a short to ground. If the circuit tests normal, replace the K9 Body Control Module.
  4. Install a 3 A fused jumper wire between the signal circuit terminal 2 and ground.
  5. Verify the scan tool BCM Traction Control Switch parameter is Active.
    6. ⇒If not the specified value, test the signal circuit for a short to voltage or an open/high resistance. If the circuit tests normal, replace the K9 Body Control Module.
  6. If all circuits test normal, test or replace the S48C Multi-Function Switch - Instrument Panel 1.

Component Testing

Static Test

  1. Ignition OFF, disconnect the harness connector at the S48C Multi-Function Switch - Instrument Panel 1.
  2. Test for infinite resistance between the signal circuit terminal 2 and the ground circuit terminal 4 with the switch in the released position.
    3. ⇒If not within the specified range, replace the S48C Multi-Function Switch - Instrument Panel 1.
  3. Test for less than 5 Ω between the signal circuit terminal 2 and the ground circuit terminal 4 with the switch in the pressed position.
    4. ⇒If not within the specified range, replace the S48C Multi-Function Switch - Instrument Panel 1.
