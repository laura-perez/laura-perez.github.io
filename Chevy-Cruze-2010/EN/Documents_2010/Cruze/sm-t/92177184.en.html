Inflatable Restraint Sensing and Diagnostic Module Scan Tool Information

Parameter

Operating Conditions

Expected Value

Definition

Operating Conditions: Ignition ON

Automatic Occupant Sensing Current Front Occupancy Status

 

Empty

The scan tool displays Unknown, Empty, Occupied, or Reserved. This is the state of the passenger seat if it is occupied or empty and monitored by the passenger presence system.

Automatic Occupant Sensing Fault Status

 

No Fault

The scan tool displays No Fault, Fault Present, or Vehicle Mismatch. This is the internal status of the passenger presence system monitored by the SDM.

Automatic Occupant Sensing System Enabled

 

Enabled

The scan tool displays if the passenger presence system is enabled.

Automatic Occupant Sensing System Learned

 

Learned

The scan tool displays Learned or Unlearned. Learned is displayed if the passenger presence is properly programmed.

Base Model Part Number

 

Varies

The scan tool displays the part number of the SDM which is stored in non volatile memory.

Driver Seat Belt Sensor Enabled

 

Enabled

The scan tool displays if the seat belt sensor in the seat belt buckle is enabled.

Driver Seat Belt Sensor Learned

 

Learned

The scan tool displays Learned or Unlearned. Learned is displayed if the SDM is properly programmed for the driver seat belt sensor.

Driver Seat Position Sensor Learned

 

Learned

The scan tool displays Learned or Unlearned. Learned is displayed if the SDM is properly programmed for the driver seat position sensor.

Driver Seat Forward Position Switch Status

 

Varies

The scan tool displays Forward or Rearward. This parameter indicates the position of the driver seat.

Driver Seat Position Sensor Enabled

 

Enabled

The scan tool displays if the seat position sensor of the driver seat is enabled.

End Model Part Number

 

Varies

The scan tool displays the end model part number. The primary key is the last four digits of the end model number. The end model number is unique to each calibration/configuration and it is stored in non volatile memory of the SDM.

Fasten Driver Seat Belt Indicator Status

 

Off

The scan tool displays if the seat belt indicator is illuminated. The Indicator will illuminate initially if the seat belt is not buckled.

Front Passenger Classification

--

Empty

The scan tool displays Unknown, Empty, Child, Small Adult, or Large Adult. This is the state of what occupies the passenger seat, as monitored by the passenger presence system.

Front Row Driver Seat Belt Attached Status

 

Unbuckled

The scan tool displays Buckled or Unbuckled. This is the state of the driver seat belt switch in the driver seat belt buckle.

Front Row Passenger Seat Belt Attached Status

 

Unbuckled

The scan tool displays Buckled or Unbuckled. This is the state of the passenger seat belt switch in the passenger seat belt buckle.

ESS 1-8 Enabled

 

Enabled

The scan tool displays Enabled or Disabled. This is the state of the sensor.

ESS 1-8 Learned

 

Learned

The scan tool displays Learned or Unlearned. Learned is displayed if the SDM has the sensor programmed.

ESS 1-8 Type

 

Varies

The scan tool displays Not Used, Front Sensor LF, Front Sensor Center, Front Sensor RF, Side Impact Sensor Left Row 1, Side Impact Sensor Right Row 1, Side Impact Sensor Left Row 2 or Side Impact Sensor Right Row 2. This is the type of SIR sensor.

Loop 1-18 Enabled

 

Enabled

The scan tool displays Enabled or Disabled This is the status of the inflator module where it is capable of deploying when it is enabled.

Loop 1-18 Learned

 

Learned

The scan tool displays Learned or Unlearned. Learned is displayed if the SDM has the number loop programmed with the present connected inflator module.

Loop 1-18 Resistance

 

Varies

The scan tool displays the resistance of the loop with the inflator module connected.

Loop 1-18 Type

 

Varies

The scan tool displays Not Used, LF Stage 1, RF Stage 1, LF Stage 2, RF Stage 2, LF Pretensioner, RF Pretensioner, LF Side Bag, RF Side Bag, Lt. Roof Stage 1, Rt. Roof Stage 1, Lt. Roof Stage 2, Rt. Roof Stage 2, Lt. Roof Stage 3 or Rt. Roof Stage 3. This is the type of loop.

Occupant Protection System Warning Indicator Status

  

The scan tool displays if the passenger presence system warning indicator is illuminated. The Indicator will illuminate depending if a child or an adult is sitting in the passenger seat.

Passenger Airbag Disabled

 

Disabled

The scan tool indicates if the passenger airbag is disabled.

Passenger Airbag Off Indicator Enabled

 

Enabled

The scan tool displays Enabled or Disabled This is the status of the passenger airbag off indicator where when it is enabled it is illuminated.

Passenger Airbag Off Indicator Learned

 

Learned

The scan tool displays Learned or Unlearned. Learned is displayed if the SDM has the airbag off indicator programmed.

Passenger Airbag On Indicator Learned

 

Learned

The scan tool displays Learned or Unlearned. Learned is displayed if the SDM has the airbag on indicator programmed.

Passenger Airbag Deactivation Status

 

Active

The scan tool displays if the passenger airbag is deactivated. The airbag will be deactivated if a child or child seat is present in the passenger seat.

Passenger Airbag Manual Disable Status

 

Enabled

The scan tool displays Enabled or Disabled This is the status of the passenger airbag where the airbag can be disabled manually.

Passenger Airbag On Indicator Enabled

 

Enabled

The scan tool displays Enabled or Disabled This is the status of the passenger airbag on indicator where when it is enabled it is illuminated.

Passenger Airbag On Indicator Status

 

Off

The scan tool displays if the passenger air bag on indicator is illuminated. The Indicator will illuminate depending if a child or an adult is sitting in the passenger seat.

Passenger Seat Belt Reminder Indicator Enabled

 

Enabled

The scan tool displays Enabled or Disabled This is the status of the passenger seat belt reminder indicator where when it is enabled it is illuminated.

Passenger Seat Belt Reminder Indicator Learned

 

Learned

The scan tool displays Learned or Unlearned. Learned is displayed if the SDM has the passenger seat belt indicator programmed.

Passenger Seat Belt Reminder Sensor Enabled

 

Enabled

Passenger Seat Belt Reminder Sensor Learned

Passenger Seat Belt Sensor Enabled

Passenger Seat Belt Sensor Learned

 

Learned

The scan tool displays Learned or Unlearned. Learned is displayed if the SDM is properly programmed for the passenger seat belt sensor.

Passenger Seat Forward Position Switch Status

  

The scan tool displays Forward or Rearward. This parameter indicates the position of the driver seat.

Passenger Seat Position Enabled

 

Enabled

The scan tool displays if the seat position sensor of the driver seat is enabled.

Passenger Seat Position Sensor Learned

 

Learned

The scan tool displays Learned or Unlearned. Learned is displayed if the SDM is properly programmed for the passenger seat position sensor.

SDM Primary Key

 

Varies

The scan tool displays the primary key that the SDM has stored to memory.

System Power Mode

--

Accessory

The scan tool displays Off, Accessory, Run or Crank Request. This is the state of the ignition switch.

Vehicle Identification Number

 

Varies

The VIN number of the vehicle programmed into the SDM.
