Circuit/System Description

When any exterior door handle is lifted, switched ground signals are sent to the remote control door lock receiver through the door handle switch signal circuits. Upon receipt of these inputs, the remote control door lock receiver will check the door lock status of the following inputs:

    • Door unlock switch activation
    • Keyless entry unlock command
    • Passive keyless entry unlock command message is received by the remote control door lock receiver

Once the enable criteria has been met, the remote control door lock receiver will supply power and ground to door latch control circuits and the door will release.

Conditions for Running the DTC

Battery voltage is between 9-16 V.

Conditions for Setting the DTC

B251A 01

The remote control door lock receiver detects a short to voltage in any door unlatch motor unlatch control low circuit.

B251A 02

The remote control door lock receiver detects a short to ground in any door unlatch motor unlatch control low circuit.

B251B 04

The remote control door lock receiver detects an open in the driver door unlatch motor unlatch control low or high circuit.

B251C 04

The remote control door lock receiver detects an open in the passenger door unlatch motor unlatch control low or high circuit.

B251D 02

The remote control door lock receiver detects an open in the left rear door unlatch motor unlatch control low or high circuit.

B251E 04

The remote control door lock receiver detects an open in the right rear door unlatch motor unlatch control low or high circuit.

Action Taken When the DTC Sets

The remote control door lock receiver ignores inputs from the exterior door handle switches.

Conditions for Clearing the DTC
    • The DTC will be current for as long as the fault is present.
    • When the fault is no longer present, the DTC will be a history DTC.
    • A history DTC will clear after 50 ignition cycles.

Circuit/System Verification

Ignition ON, command the Driver Door Unlatch and Passenger Door(s) Unlatch with the scan tool. The driver and passenger door latches should latch and unlatch when changing between commanded states.

Circuit/System Testing
  1. Ignition OFF, disconnect the harness connector at the malfunctioning door latch assembly.
  2. Ignition ON, verify that a test lamp does not illuminate between control circuit terminal 5 and ground.
    3. ⇒If the test lamp illuminates, test the control circuit for a short to voltage. If the circuit tests normal. replace the K77 Remote Control Door Lock Receiver.
  3. Verify that a test lamp does not illuminate between control circuit terminal 10 and ground.
    4. ⇒If the test lamp illuminates, test the control circuit for a short to voltage. If the circuit tests normal. replace the K77 Remote Control Door Lock Receiver.
  4. Verify that a test lamp does not illuminate between control circuit terminal 5 and B+.
    5. ⇒If the test lamp illuminates, test the control circuit for a short to ground. If the circuit tests normal. replace the K77 Remote Control Door Lock Receiver.
  5. Verify that a test lamp does not illuminate between control circuit terminal 10 and B+.
    6. ⇒If the test lamp illuminates, test the control circuit for a short to ground. If the circuit tests normal. replace the K77 Remote Control Door Lock Receiver.
  6. Connect a test lamp between the control circuit terminals 5 and 10.
  7. Command the Driver Door Unlatch and Passenger Door(s) Unlatch with a scan tool. Verify the test lamp illuminates briefly, when changing between the commanded states.
    8. ⇒If the test lamp does not illuminate, test the control circuits for an open/high resistance. If the circuits test normal, replace K77 Remote Control Door Lock Receiver.
  8. If all circuits test normal, test or replace the door latch assembly.

Component Testing

Door Latch Assembly Test

  1. Ignition OFF, disconnect the harness connector at the appropriate door latch assembly.
  2. Ignition ON, install a 15 A fused jumper wire between the control terminal listed below and B+:
    3. • Left hand drive: Driver door latch control terminal 10
    • Left hand drive: Passenger door latch control terminal 5
    • Right hand drive: Driver door latch control terminal 5
    • Right hand drive: Passenger door latch control terminal 10
    • Left rear door latch control terminal 10
    • Right rear door latch control terminal 5
  3. Momentarily install a jumper wire between the control terminal listed below and ground. The door latch should perform the Unlock function:
    4. • Left hand drive: Driver door latch control terminal 5
    • Left hand drive: Passenger door latch control terminal 10
    • Right hand drive: Driver door latch control terminal 10
    • Right hand drive: Passenger door latch control terminal 5
    • Left rear door latch control terminal 5
    • Right rear door latch control terminal 10
    ⇒If the function does not perform as specified, replace the door latch assembly.
  4. Reverse the jumper wires; the door latch assembly should perform the Lock function.
    5. ⇒If the function does not perform as specified, replace the door latch assembly.
