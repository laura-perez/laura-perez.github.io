Crankshaft and Bearing Installation

Special Tools

EN-45059 Torque Angle Sensor Kit



    Note: Inspect the installation position.

  1. Install the crankshaft bearing clips, oil bearing clips.
  2. Install the crankshaft.

    3. Note: Inspect the installation position.

  3. Install the crankshaft bearing caps 1-4.
    4. • Oil bearing clips.
    • Install the 8 bolts.

    Note: Inspect the installation position.

  4. Install the crankshaft bearing cap.
    5. 4.1. Apply black adhesive sealing compound to the grooves of the rear crankshaft bearing cap.
    4.2. Install the 2 bolts.

    Caution: Refer to Fastener Caution in the Preface section.



  5. Tighten the crankshaft bearing cap (1).
  6. Install 10 NEW bolts and tighten in 3 passes use the EN-45059 sensor kit:
    7. • First pass to  50 N·m (37 lb ft).
    • Second pass to  45 degrees.
    • Third pass to  15 degrees.


    Note: Inspect the installation position.

  7. Install the connecting rod bearing caps 2 and 3 (1) and oil the bearing clips.
  8. Tighten the 4 connecting rod bearing cap bolts in 3 passes use the EN-45059 sensor kit:
    9. • First pass to  35 N·m (26 lb ft)
    • Second pass to  45 degrees
    • Third pass to  15 degrees


  9. Turn the crankshaft through 180 degrees.

    10. Note: Inspect the installation position.

  10. Install the connecting rod bearing caps 1 and 4 (1) and oil the bearing clips.
  11. Tighten the 4 connecting rod bearing cap bolts in 3 passes use the EN-45059 sensor kit:
    12. • First pass to  35 N·m (26 lb ft)
    • Second pass to  45 degrees
    • Third pass to  15 degrees
