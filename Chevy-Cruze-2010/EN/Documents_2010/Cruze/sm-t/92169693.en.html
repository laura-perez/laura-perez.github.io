Cylinder Head Installation

Special Tools

EN-45059 Torque Angle Sensor Kit

For equivalent regional tools, refer to Special Tools .



    Note: Ensure that the engine block and cylinder head surface are clear of oil, grease and old gasket material.

  1. Install a NEW cylinder head gasket (2).
  2. Install the cylinder head (3).

    3. Caution: Refer to Fastener Caution in the Preface section.



    Note: Always use new cylinder head bolts.

  3. Install the 10 NEW cylinder head bolts.
  4. Tighten the 10 cylinder head bolts in a sequence as shown in the graphic.
  5. Tighten the 10 NEW cylinder head bolts in three passes:
    6. 5.1. First pass in sequence to  65 N·m (48 lb ft).
    5.2. Second pass in sequence to an additional  120 degrees using the EN-45059 angle sensor .
    5.3. Third pass in sequence to an additional  120 degrees using the EN-45059 angle sensor .


  6. Install the 8 valve lifter bridges (2, 3)
  7. Install the 8 valve rocker arms (4, 1).
