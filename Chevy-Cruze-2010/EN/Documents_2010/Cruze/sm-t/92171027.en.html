Ignition Coil Installation

Special Tools

EN 49424 Remover and Installer Ignition Module

For equivalent regional tools, refer to Special Tools .



    Note: When installing a new DIS ignition module it is not necessary to replace the gaskets.

  1. Install the EN 49424 remover / installer .
  2. Install the ignition coil (1) and the NEW gaskets (2).

    3. Caution: Refer to Fastener Caution in the Preface section.

    Note: Note the arrow on the cover.

  3. Install the 2 ignition coil bolts an tighten to  8 N·m (71 lb in).
  4. Install the cover of the ignition coil in the direction of the arrow.
