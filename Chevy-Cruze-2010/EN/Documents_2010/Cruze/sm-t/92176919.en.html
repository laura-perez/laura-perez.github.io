Inflatable Restraint Passenger Presence System Scan Tool Information

Parameter

Operating Conditions

Expected Value

Definition

Operating Conditions: Ignition is ON, engine OFF, and vehicle is stationary.

Battery Voltage

--

Volts

This is the voltage input supplied to the passenger presence system.

Passenger Presence Classification

--

Empty

The scan tool displays Unknown, Empty, Child, Small Adult, or Large Adult. This is the state of what occupies the passenger seat, as monitored by the passenger presence system.

Passenger Presence Detection

--

Empty

The scan tool displays Unknown, Empty, or Occupied. This is the state of the passenger seat if it is occupied or empty as monitored by the passenger presence system.

Passenger Presence Detection System Status

--

No Fault

The scan tool displays No Fault, Plant Mode, Fault Mismatch, or Fault Barrier. This is the internal status of the operating mode of the passenger presence system.

Passenger Seat Belt Status

--

Unbuckled

The scan tool displays Buckled or Unbuckled. This is the state of the seat belt switch in the seat belt buckle.

Primary Key Status

--

Valid

The scan tool displays Unknown, Valid or Invalid. This is the state of the primary key programmed in the passenger presence system module.

Power Mode

--

Accessory

The scan tool displays Off, Accessory, Run or Crank Request. This is the state of the ignition switch.
