Digital Audio Broadcast Detection Module Output Controls

Scan Tool Output Control

Description

Audio Tone

This output controls a test tone (1 kHz sinus) on left and right audio channels in order to check the audio between the Digital Audio Broadcast Module and the radio. Pressing left button a tone with 1 KHz can be heard out off the left speakers. Pressing right button a tone with 1 KHz can be heard out off the right speaker.

CAN Bus Configuration Learn

This output control will enable the module to learn the serial data speed and adapt to it.
