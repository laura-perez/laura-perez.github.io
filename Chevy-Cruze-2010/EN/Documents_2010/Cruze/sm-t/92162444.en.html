1. Air Recirculation Malfunction

Circuit/System Description

The desired position of the recirculation flap can be set with the recirculation switch at the HVAC control. The selected value is passed to the HVAC control module via the LIN-Bus. The HVAC control module supplies a 12 V reference voltage to the stepper motor and energizes the 4 stepper motor coils with a pulsed ground signal. The stepper motor puts the recirculation flap into the selected position.

Circuit/System Testing

  1. Ignition ON, activate and deactivate the defrost switch. Verify the scan tool Enhanced Defrost Switch parameter changes between Active and Inactive.
    2. ⇒If not the specified value, replace the A26 HVAC Control.
  2. Ignition ON, activate and deactivate the recirculation switch. Verify the scan tool Air Recirculation Mode Switch parameter changes between Active and Inactive.
    3. ⇒If not the specified value, replace the A26 HVAC Control.
  3. Ignition ON, activate and deactivate the AQS switch. Verify the scan tool AQS (Air Quality Sensor) Switch parameter changes between Active and Inactive.
    4. ⇒If not the specified value, replace the A26 HVAC Control.
  4. If all circuits test normal, replace the M46 Recirculation Actuator and verify the M46 Recirculation Actuator can be activated and deactivated. If the M46 Recirculation Actuator cannot be activated or deactivated, replace the K33 HVAC Control Module.
