Instrument Cluster Scan Tool Data Parameters

Parameter

Expected Value

Description

Operating Conditions: Ignition ON

Ambient Air Temperature

15 °C

The scan tool displays outside temperature in °C.

Battery Voltage

12-14 V

The scan tool displays Volts. This is the current battery voltage signal.

Coolant Level Switch

Open

The scan tool displays either Open, Closed, or Not Available.

Driver Information Center Reset Switch

Active/Inactive

The scan tool displays Active or Inactive. This is the state of the driver information center reset switch.

Driver Information Center Switch

Varies

The scan tool displays Volts. This is the current driver information center switch input voltage.

Driver Information Center Switch

Inactive

The scan tool displays either Inactive, Up, Down, or Menu. This is the state of the driver information center switch.

Key in Ignition Status

Active/Inactive

The scan tool displays Active or Inactive. This is the state of the key in ignition switch.

Reset Switch

Active/Inactive

The scan tool displays Active or Inactive. This is the state of the reset switch.

Washer Fluid Heater

Off

The scan tool displays either Off, On, or Not Available.

Washer Fluid Level Switch

Open

The scan tool displays either Open, Closed, or Not Available.

Instrument Cluster Scan Tool Output Controls

Output Control

Description

All Indicators

This function is used to command all the indicators ON or OFF.

CAN Bus Configuration Learn

This function is used to Reset the source ID for low speed CAN Bus communications.

Driver Information Center Dimming

This function is used to command the dimming of the driver information display to Increase or Decrease by 10%.

Driver Information Center Options

This function is used to command the driver information center buttons using the scan tool.

Driver Information Center Segments

This function is used to command the driver information center segments ON and OFF.

Instrument Cluster Gauge Sweep

This function is used to command the gauges to sweep from minimum to maximum.

Instrument Panel Cluster Dimming

This function is used to command the dimming of the instrument cluster to Increase or Decrease by 10%.

Odometer Programming

This function is used to command the odometer to reset.
