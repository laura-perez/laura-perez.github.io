Circuit/System Description

The radio or the chime module, if not equipped with a radio, generates the audible warnings. The instrument cluster, the body control module (BCM), the inflatable restraint sensing and diagnostic module (SDM) or the object alarm module request audible warnings via Low Speed CAN-Bus signals.

Circuit/System Verification

Note: Before performing this diagnostic, make sure no indicators are illuminated after the instrument cluster performs a lamp test. If any indicators are illuminated after the bulb test, perform the indicator diagnostics before this diagnostics.

Without Radio

Ignition ON, perform the Chime test with the scan tool, verify that all audible warnings sound normal.

⇒If the audible warnings do not operate properly, replace the P5 Chime Module.

With Radio

  1. Ignition ON, radio ON, adjust the radio balance and fade to each speaker.
    2. ⇒If any speaker does not operate properly, refer to Speaker Malfunction .
  2. Ignition ON, buckle up driver seat belt, verify the scan tool Driver Seat Belt Status parameter is Buckled.
    3. ⇒If not Buckled refer to Seat Belt Indicator Circuit Malfunction - Driver .
  3. Ignition ON, headlamp switch OFF, verify the scan tool Headlamp On Switch parameter is Inactive.
    4. ⇒If not Inactive, refer to Headlamps Malfunction .
  4. Ignition ON, parking lamp switch OFF, verify the scan tool Parklamp Switch parameter is Inactive.
    5. ⇒If not Inactive, refer to Park, License, and/or Tail Lamps Malfunction .
  5. Ignition ON, turn signal switch OFF, verify the scan tool Left Turn Signal Switch parameter and the scan tool Right Turn Signal Switch parameter is Inactive.
    6. ⇒If not Inactive, refer to Turn Signal Lamps and/or Indicators Malfunction .
  6. Ignition ON, parking brake released, verify the scan tool Park Brake Switch parameter is Inactive.
    7. ⇒If not Inactive, refer to Park Brake System Diagnosis .
  7. Ignition ON, doors closed, verify that all scan tool parameters listed below are Inactive.
    8. • Driver Door Ajar Switch
    • Passenger Door Ajar Switch
    • Left Rear Door Ajar Switch
    • Right Rear Door Ajar Switch
    ⇒If any are not Inactive, refer to Door Ajar Indicator Malfunction .
  8. Verify the scan tool Key in Ignition Status parameter is Inactive when the key is in the ignition and Active when the key is out of the ignition.
    9. ⇒If the scan tool parameter does not operate properly, replace the K9 Body Control Module (BCM).
  9. Ignition ON, verify that the scan tool Park Assist Chime parameter is Active when the parking assist is active and Inactive when the parking assist is Inactive.
    10. ⇒If the scan tool parameter does not operate properly, refer to Parking Assist System Malfunction .
  10. If all tests operate properly, replace the A11 Radio.
