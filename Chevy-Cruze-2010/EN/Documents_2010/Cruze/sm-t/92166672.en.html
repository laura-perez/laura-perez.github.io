Driver or Passenger Seat Back Cushion Cover and Pad Replacement

Removal Procedure

Warning: Refer to SIR Warning in the Preface section.

Warning: Replace the passenger presence system as a complete assembly to prevent possible injury to the occupant. The bladder, the pressure sensor, the seat cushion, and the control module are assembled and calibrated as a unit. Using only some of the components in the service kit will cause the passenger presence system to operate improperly.

  1. Remove front seat adjuster, if equipped. Refer to Front Seat Adjuster Replacement .
  2. Remove driver or passenger seat. Refer to Driver or Passenger Seat Replacement .
  3. Remove seat cushion inner trim panel. Refer to Front Seat Cushion Inner Trim Panel Replacement .
  4. Remove seat cushion outer trim panel. Refer to Front Seat Cushion Outer Trim Panel Replacement .
  5. Remove front seat head restraint guide. Refer to Front Seat Head Restraint Guide Replacement .


  6. Detach cushion cover from backrest below.
  7. Release and disconnect wiring harness plug (1), if equipped.
  8. Remove 4 front seat backrest screw (2).
  9. Remove backrest.


  10. Remove cushion cover on backside.
    11. • Open 2 zip.
    • Remove 8 cushion covering.
  11. Remove front seat back cushion cover.
    12. • Detach 4 from frame.
    • Remove cushion cover from Velcro.
  12. Remove front seat back cushion pad.

Installation Procedure


  1. Install front seat back cushion pad.
  2. Install front seat back cushion cover.
    3. • Install cushion cover to Velcro.
    • Attach 4 to frame.
  3. Install cushion cover to backside.
    4. • Install 8 cushion covering
    • Close 2 zip.


  4. Install backrest.

    5. Caution: Refer to Fastener Caution in the Preface section.

  5. Install the front seat backrest bolts (2) and tighten to  XX N·m (XX lb in).
  6. Connect and latch wiring harness plug (1), if equipped.
  7. Attach cushion cover to backrest below.
  8. Install front seat head restraint guide. Refer to Front Seat Head Restraint Guide Replacement .
  9. Install seat cushion outer trim panel. Refer to Front Seat Cushion Outer Trim Panel Replacement .
  10. Install seat cushion inner trim panel. Refer to Front Seat Cushion Inner Trim Panel Replacement .
  11. Install driver or passenger seat. Refer to Driver or Passenger Seat Replacement .
  12. Install front seat adjuster, if equipped. Refer to Front Seat Adjuster Replacement .
