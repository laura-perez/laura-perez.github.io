Circuit/System Description

The body control module (BCM) monitors, receives and transmits serial data to each of the doors switches and window motors. When a window switch has been activated on the driver window switch, a signal is sent to the BCM which sends a signal to the appropriate window motor and the window motor will move in the direction commanded.

Conditions for Running the DTC

Battery voltage must be between 9-16 V.

Conditions for Setting the DTC

B316B 00

The BCM has detected the driver right front or left rear or right rear window switch is active for greater than 60 seconds.

B316B 02

The BCM has detected the driver left front window switch is active for greater than 60 seconds.

B317A 02

The BCM has detected the passenger window switch is active for greater than 60 seconds.

B318A 02

The BCM has detected the left rear window switch is active for greater than 60 seconds

B319A 02

The BCM has detected the right rear window switch is active for greater than 60 seconds

Action Taken When the DTC Sets

The affected power window will malfunction.

Conditions for Clearing the DTC

B316B 00, B316B 02, B317A 02, B318A 02, B319A 02

    • The DTC will be current for as long as the fault is present.
    • The DTC will change from current to history when the fault is no longer present.
    • A history DTC will clear after 100 consecutive ignition cycles if the condition for the malfunction is no longer present.

B316B 39

If the DTC is current, the driver window switch must be replaced before the DTC can be cleared.

Circuit/System Verification
  1. Ignition ON, command each window UP and DOWN by using the left front, right front, right rear, and left rear switches on the S79D window switch-driver. Each window should perform the UP and DOWN function from each switch.
    2. ⇒If a window does not perform the UP or DOWN function, refer to Driver Window Switch Malfunction.
  2. Command the passenger, left rear and right rear windows UP and DOWN by using the S79P window switch-passenger, S79RR window switch-right rear, S79LL window switch-left rear. Each window should perform the UP and DOWN function from each switch.
    3. ⇒If a window does not perform the UP or DOWN function, refer to Passenger/Rear Window Switch Malfunction.

Circuit/System Testing

Driver Window Switch Malfunction

Ignition ON, observe the scan tool Driver Left Front Window Switch, Driver Express Master Switch Active, Driver Up Master Switch Active, Front Passenger Down Master Switch Active, Front Passenger Express Master Switch Active, Front Passenger Up Master Switch Active, Left Rear Down Master Switch Active, Left Rear Express Master Switch Active, Left Rear Up Master Switch Active, Right Rear Down Master Switch Active, Right Rear Express Master Switch Active, and Right Rear Up Master Switch Active parameters while using each switch in the S79D window switch-driver. The parameter should change from Inactive to Active.

⇒If a value does not change, replace the S79D window switch-driver.

Passenger/Rear Window Switch Malfunction

Observe the scan tool Passenger Window Switch at Door, Left Rear Window Switch at Door and Right Rear Window Switch at Door parameters while using the S79P window switch-passenger, S79LR window switch-left rear and S79RR window switch-right rear. The parameter should change to Up, Down and Express.

⇒If a value does not change, replace the appropriate S79 window switch.
