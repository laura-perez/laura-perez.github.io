Circuit/System Description

When the ignition switch is in the RUN or ACCY position, or retained accessory power (RAP) is enabled and the driver or passenger window switch or rear window switch is pressed in the down position, voltage is applied to the appropriate window motor causing that window to open. When the driver or passenger window switch is pulled in the up position, voltage is applied to the appropriate window motor in the opposite direction causing that window to close.

Circuit/System Verification
  1. Ignition ON, command the driver, passenger, left rear and right rear windows UP and DOWN by using the S79D window switch-driver, S79P window switch-passenger, S79LR window switch-left rear and S79RR window switch-right rear. Each window should perform the UP and DOWN function from each switch.
    2. ⇒If the driver window malfunctions, refer to Driver Window Malfunction.
    ⇒If the passenger, left rear or right rear window malfunctions, refer to Passenger/Rear Window Malfunction.
  2. Command each window to EXPRESS DOWN by pressing appropriate S79 window switch past the first detent and releasing. Each window should perform the EXPRESS DOWN function.
    3. ⇒If a window does not perform EXPRESS DOWN function, replace the S79 window switch.

Circuit/System Testing

Driver Window Malfunction

  1. Ignition OFF, disconnect the harness connector at the S79D window switch-driver.
  2. Test for less than 15 Ω between the ground circuit terminal 1 and ground.
    3. ⇒If greater than the specified range, test the ground circuit for an open/high resistance.
  3. Ignition ON, verify that a test lamp illuminates between the B+ circuit terminal 4 and ground.
    4. ⇒If the test lamp does not illuminate, test the B+ circuit for a short to ground or an open/high resistance.
  4. Ignition OFF, connect the harness connector at the S79D window switch-driver. Disconnect the harness connector at the M74D window motor-driver.
  5. Connect a test lamp between the control circuit terminal A and control circuit terminal B.
  6. Ignition ON, command the M74D window motor-driver UP and DOWN by using the S79D window switch-driver. The test lamp should illuminate during the UP and DOWN commands.
    7. ⇒If the test lamp does not illuminate during the UP or DOWN command, test either control circuit for a short to ground or an open/high resistance. If the circuits tests normal, test or replace the S79D window switch-driver.
    ⇒If the test lamp is always ON, test either control circuit for a short to voltage. If the circuits test normal, test or replace the S79D window switch-driver.
  7. If all circuits test normal, test or replace the M74D window motor-driver.

Passenger/Rear Window Malfunction

  1. Ignition OFF, disconnect the harness connector at the appropriate S79 window switch.
  2. Test for less than 15 Ω between the ground circuit terminal listed below and ground:
    3. • Passenger window switch terminal 1
    • Right rear window switch terminal 5
    • Left rear window switch terminal 3
    • Left rear window switch terminal 5
    ⇒If greater than the specified range, test the ground circuit for an open/high resistance.
  3. Ignition ON, verify that a test lamp illuminates between the B+ circuit terminal listed below and ground:
    4. • Passenger window switch terminal 4
    • Right rear window switch terminal 8
    • Left rear window switch terminal 8
    ⇒If the test lamp does not illuminate, test the B+ circuit for a short to ground or an open/high resistance.
  4. Ignition OFF, connect the harness connector at the S79 window switch and disconnect the harness connector at the M74 window motor.
  5. Connect a test lamp between the control circuit terminal A and control circuit terminal B.
  6. Ignition ON, command the M74 window motor UP and DOWN by using the S79 window switch. The test lamp should briefly illuminate during the UP and DOWN commands.
    7. ⇒If the test lamp does not illuminate during the UP or DOWN command, test either control circuit for a short to ground or an open/high resistance. If the circuits tests normal, test or replace the S79 window switch.
    ⇒If the test lamp is always ON, test either control circuit for a short to voltage. If the circuits test normal, test or replace the S79 window switch.
  7. If all circuits test normal, test or replace the M74 window motor.

Component Testing

Window Motor

  1. Ignition OFF, disconnect the harness connector at the appropriate M74 window motor.
  2. Install a 25A fused jumper wire between one of the control terminals and 12V. Momentarily install a jumper wire between the other control terminal and ground. Reverse the jumper wires at least two times, the M74 window motor should perform the UP/DOWN functions.
    3. ⇒If it does not perform as specified, replace the M74 window motor.
