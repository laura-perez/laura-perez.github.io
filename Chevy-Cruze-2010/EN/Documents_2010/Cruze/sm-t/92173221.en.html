Circuit/System Description

The passenger air bag ON/OFF indicators are used to notify the driver when the passenger presence system has enabled or disabled the I/P inflator module. The sensing and diagnostic module (SDM) controls all of the indicator control circuits to the passenger air bag ON/OFF indicators. When the passenger presence system enables or disables the I/P inflator module it communicates with the SDM through serial data requesting illumination of the indicators.

Circuit/System Verification

    Note: The passenger air bag ON and OFF indicators may not operate as expected if there are supplemental inflatable restraint (SIR) DTCs. Repair all SIR DTC before proceeding with this diagnostic.

  1. Ignition ON, the passenger air bag ON and OFF indicators should both illuminate for approximately 5 seconds for a bulb check.
  2. Monitor the parameter Passenger Presence Detection Occupancy Status with a scan tool. With the passenger seat unoccupied, the parameter should display Empty. The passenger air bag OFF indicator should illuminate.
  3. Apply enough weight to change the parameter display to Occupied. The passenger air bag indicator should switch from OFF to ON within 5 seconds.

Circuit/System Testing

Note: When removing connectors inspect for damage or corrosion. Damage or corrosion in the following requires repair or replacement of the affected.

   • Inflatable restraint passenger air bag on/off indicator.
   • Inflatable restraint passenger air bag on/off indicator wiring harness.

  1. Ignition OFF, disconnect the harness connector at the P14 inflatable restraint passenger I/P module indicator.
  2. Connect a test lamp between a ground and the control circuit terminal 2.
  3. With the passenger seat unoccupied, then occupied the test lamp should switch states within 5 seconds.
    4. ⇒If the test lamp is always on, test the control circuit for a short to ground. If the circuit tests normal replace the K36 SDM.
    ⇒If the test lamp is always off, test the control circuit for a short to voltage or an open/high resistance. If the circuit tests normal replace the K36 SDM.
  4. Connect a test lamp between a ground and the control circuit terminal 5.
  5. With the passenger seat unoccupied, then occupied the test lamp should switch states within 5 seconds.
    6. ⇒If the test lamp is always on, test the control circuit for a short to ground. If the circuit tests normal replace the K36 SDM.
    ⇒If the test lamp is always off, test the control circuit for a short to voltage or an open/high resistance. If the circuit tests normal replace the K36 SDM.
  6. If all circuits test normal, replace the P14 inflatable restraint passenger air bag ON/OFF indicator.
