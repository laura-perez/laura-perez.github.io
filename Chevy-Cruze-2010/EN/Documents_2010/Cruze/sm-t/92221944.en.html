DTC Descriptor

DTC P0315 00: Crankshaft Position System Variation Not Learned

Circuit/System Description

The crankshaft position sensor system variation learn feature is used to calculate reference period errors caused by slight build tolerance variations in the crankshaft, and crankshaft position sensor position. The calculated error allows the engine control module (ECM) to accurately compensate for reference period variations. This enhances the ability of the ECM to detect misfire events over a wide range of engine speeds and load conditions. The ECM stores the crankshaft position system variation values after a learn procedure has been performed. This enhances the ability of the ECM to detect misfire events over a wider range of engine speed and load. If the actual crankshaft position variation is not stored within the crankshaft position sensor compensating value look up table, then DTC P0300 00 may set. If the crankshaft position sensor system variation values are not stored in the ECM memory, or a proper ECM power down does not occur after completing the crankshaft position sensor Learn Procedure, then DTC P0315 00 sets.

Conditions for Running the DTC
    • The engine is running.
    • DTC P0315 00 runs continuously once the above conditions are met.

Conditions for Setting the DTC

The ECM detects that the crankshaft position sensor system variation values are not stored in memory.

Action Taken When the DTC Sets

DTC P0315 00 is a Type A code.

Conditions for Clearing the DTC

DTC P0315 00 is a Type A code.

Diagnostic Aids

This test procedure requires that the vehicle battery has passed a load test and is completely charged. Refer to Battery Inspection/Test .

Circuit/System Testing

Note: The B26 Crankshaft Position Sensor system variation learn procedure may have to be repeated up to 5 times before the procedure is learned.

Perform the B26 Crankshaft Position Sensor system variation learn procedure. Refer to Crankshaft Position System Variation Learn. The scan tool should display Learned this Ignition.

⇒If the crankshaft position variation learn procedure cannot be performed successfully, inspect for the following conditions:
    • Any worn crankshaft main bearings
    • A damaged or misaligned reluctor wheel
    • Excessive crankshaft runout
    • A damaged crankshaft. Refer to Crankshaft and Bearing Cleaning and Inspection
    • Interference in the signal circuit of the B26 Crankshaft Position Sensor
    • The ignition switch is left in the ON position, until the battery is discharged
    • An ECM power disconnect, with the ignition ON, that may have erased the crankshaft position sensor system variation values and set DTC P0315 00
    • Any debris between the B26 Crankshaft Position Sensor and the reluctor wheel
       ⇒If the ECM is still unable to complete the learn procedure, replace the K20 Engine Control Module.
