Circuit/System Description

The park lamps and license plate lights are turned ON when the headlamp switch is placed in the Parking lights or Low beam position or anytime the headlights are requested. When the body control module (BCM) receives an request to switch ON the park lamps and license plate lights the BCM provides a sends out a PWM signal, which illuminates the park lamps and license plate lights.

Circuit/System Verification
  1. Switch the park lamps ON and OFF using the headlamp switch. Verify the scan tool Park Lamp Switch parameter cycles between Active and Inactive as commanded.
    2. ⇒If the parameter does not cycle between the specified values, refer to Park Lamp Switch Malfunction.
  2. Ignition ON, switch ON and OFF the park lamps and license plate lights by using the headlamp switch. Park lamps and license plate lights should go ON and OFF.
    3. ⇒If the tail lamps are inoperative, refer to Tail Lamps Inoperative.
    ⇒If the front park lamps are inoperative, refer to Front Park Lamps Inoperative.
    ⇒If the license plate lights are inoperative, refer to License Plate Lights Inoperative.

Circuit/System Testing

Tail Lamps Inoperative

  1. Ignition OFF, disconnect the harness connector at the appropriate inoperative tail lamp.
  2. Test for less than 5 Ω between the appropriate tail lamp ground circuit harness connector listed below and ground.
    3. • X910 Body Harness & Tail Lamp Left Harness connector terminal 4
    • X911 Body Harness & Tail Lamp Right Harness connector terminal 4
    • X914 Liftgate Harness & Liftgate Tail Lamp Left Harness terminal B
    • X916 Liftgate Harness & Liftgate Tail Lamp Right Harness terminal B
    ⇒If greater than the specified value, test the ground circuit for an open/high resistance.
  3. Connect a test lamp between the appropriate tail lamp control circuit harness connector listed below and ground.
    4. • X910 Body Harness & Tail Lamp Left Harness connector terminal 2
    • X911 Body Harness & Tail Lamp Right Harness connector terminal 2
    • X914 Liftgate Harness & Liftgate Tail Lamp Left Harness terminal A
    • X916 Liftgate Harness & Liftgate Tail Lamp Right Harness terminal B
  4. Ignition ON, switch ON and OFF the tail lamps by using the headlamp switch. The test lamp should turn ON and OFF when changing between the ON/OFF states.
    5. ⇒If the test lamp is always ON, test the control circuit for a short to voltage. If the circuit tests normal, replace the K9 Body Control Module.
    ⇒If the test lamp is always OFF, test the control circuit for a short to ground or an open/high resistance. If the circuits test normal, replace the K9 Body Control Module.
  5. If all the circuits test normal, replace the appropriate inoperative tail lamp.

Front Park Lamps Inoperative

  1. Ignition OFF, disconnect the harness connector at the appropriate inoperative tail lamp.
  2. Test for less than 5 Ω between the appropriate tail lamp ground circuit harness connector listed below and ground.
    3. • M29L Headlamp Leveling Actuator - Left harness connector terminal 1
    • M29R Headlamp Leveling Actuator - Right harness connector terminal 1
    ⇒If greater than the specified value, test the ground circuit for an open/high resistance.
  3. Connect a test lamp between the appropriate tail lamp control circuit harness connector listed below and ground.
    4. • M29L Headlamp Leveling Actuator - Left harness connector terminal 1
    • M29R Headlamp Leveling Actuator - Right harness connector terminal 1
  4. Ignition ON, switch ON and OFF the tail lamps by using the headlamp switch. The test lamp should turn ON and OFF when changing between the ON/OFF states.
    5. ⇒If the test lamp is always ON, test the control circuit for a short to voltage. If the circuit tests normal, replace the K9 Body Control Module.
    ⇒If the test lamp is always OFF, test the control circuit for a short to ground or an open/high resistance. If the circuits test normal, replace the K9 Body Control Module.
  5. If all the circuits test normal, replace the appropriate inoperative park lamp.

License Plate Lights Inoperative

  1. Ignition OFF, disconnect the harness connector at the E7 License Plate Lamp.
  2. Test for less than 5 Ω between the E7 License Plate Lamp ground circuit terminal 1 and ground.
    3. ⇒If greater than the specified value, test the ground circuit for an open/high resistance.
  3. Connect a test lamp between the E7 License Plate Lamp control circuit terminal 2 and ground.
  4. Switch ON the parking lamps or low beam. The test lamp should be ON.
    5. ⇒If the test lamp is always ON, test the control circuit for a short to voltage. If the circuit tests normal, replace the K9 Body Control Module.
    ⇒If the test lamp is always OFF, test the control circuit for a short to ground or an open/high resistance. If the circuits test normal, replace the K9 Body Control Module.
  5. If all the circuits test normal, replace the E7 License Plate Lamp.

Park Lamp Switch Malfunction

  1. Ignition OFF, disconnect the harness connector at the S30 Headlamp Switch.
  2. Verify that a test lamp illuminates between the ground circuit terminal 6 and B+.
    3. ⇒If the test lamp does not illuminate, test the ground circuit for an open/high resistance.
  3. Ignition ON, test for B+ between the park lamp signal circuit terminal 3 and ground.
    4. ⇒If less than the specified value, test the signal circuit for a short to ground or an open/high resistance. If the circuit tests normal, replace the K9 Body Control Module.
  4. Install a 3 A fused jumper wire between the park lamp ON signal circuit terminal 3 and ground. Verify the scan tool Parklamp Switch parameter changes from Inactive to Active.
    5. ⇒If always Inactive, test the signal circuit for a short to voltage or an open/high resistance. If the circuit tests normal, replace the K9 Body Control Module.
    ⇒If always active, test the signal circuit for a short to ground. If the circuit tests normal, replace the K9 Body Control Module.
  5. If all circuits test normal, replace the S30 Headlamp Switch.
